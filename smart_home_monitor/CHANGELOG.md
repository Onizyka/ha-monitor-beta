# Changelog

## 1.5.0

### 🖼 Картинки устройств
* Автоматическое скачивание с zigbee2mqtt.io при обнаружении устройства
* Хранение на сервере по модели — не теряются при перезагрузке страницы
* Ручная загрузка своих картинок через модал устройства
* Одна картинка применяется ко всем устройствам одной модели

### 📊 Пороги и Telegram уведомления
* Переключатель Telegram под каждой метрикой, по умолчанию выключен
* При включении Telegram — мгновенная проверка текущего значения
* Два отдельных поля текста сообщений — для превышения Макс и для выхода ниже Мин
* Переменные в тексте: `{value}` `{unit}` `{min}` `{max}` `{name}` `{time}`
* Кнопка ↺ для возврата текста сообщения по умолчанию
* Настройка интервала проверки порогов в разделе настроек (по умолчанию 5 мин)

### ⚡ Производительность
* Кэш метрик 60 сек в памяти — быстрое открытие модала устройства
* Предзагрузка метрик в фоне при открытии страницы

## 1.1.0 — Первый стабильный релиз
Основные возможности

* 📡 Мониторинг Zigbee устройств — статус онлайн/офлайн, LQI, батарея
* 🔋 Заряд батарей — график разряда, пагинация
* ⚡ Токопотребление — мощность, ток, напряжение, энергия с графиком
* 📈 Конструктор графиков — любые метрики, период от 3 часов до 30 дней
* 🔔 История уведомлений — пагинация, автоудаление через 24ч
* 📨 Telegram — алерты офлайн, батареи, превышения порогов, ежедневный отчёт
* 🖼 Картинки устройств из Zigbee2MQTT или свои
