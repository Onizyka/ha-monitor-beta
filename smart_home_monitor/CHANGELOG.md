## 1.1.0-beta6 — Исправлено ожидание MariaDB и provisioning

### Исправлено
- ⏱️ **Ожидание MariaDB** — заменено с попытки подключения как root на TCP-проверку порта: аддон больше не зависает 60 секунд из-за недоступного root

- `db_root_user` — имя admin-пользователя MariaDB для создания БД (по умолчанию `root`, тип `str?`)

## 1.1.0-beta2 — Auto DB provisioning

### Новое
- 🗄️ **Автосоздание базы данных** при первом запуске аддона — если указан `db_root_password`, аддон сам создаст базу, пользователя и выдаст права
- Логика идемпотентна: при повторном запуске проверяет доступность БД и пропускает provisioning если всё уже настроено
- При ошибке (root недоступен, нет пароля) — выводит в лог точные SQL-команды для ручного создания и продолжает запуск

### Конфигурация
Новый необязательный параметр `db_root_password` — нужен только для первоначального создания БД:
```yaml
db_root_password: "your_mariadb_root_password"
```
Если параметр не указан и БД уже существует — аддон работает как прежде.

## 1.1.0 — Первый стабильный релиз

### Основные возможности
- 📡 Мониторинг Zigbee устройств — статус онлайн/офлайн, LQI, батарея
- 🔋 Заряд батарей — график разряда, пагинация
